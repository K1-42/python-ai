------------------------------------------------------------
０）準備
------------------------------------------------------------

①githubでアカウント作成
②リモートリポジトリ作成

------------------------------------------------------------
１）リモートリポジトリからローカルリポジトリを作成
------------------------------------------------------------

cd ｛ローカルリポジトリを作成したいフォルダ｝
git clone https://github.com/K1-42/python-ai.git

※URLはgithubの<Code>からコピー

------------------------------------------------------------
２）ローカルリポジトリのソース追加＆コミット
------------------------------------------------------------

①ソース追加・更新
cd｛ローカルリポジトリを作成したフォルダ｝
git add git_command.txt

（まとめて追加・更新）git add .


②状態確認（更新状態とかも分かる）
git status

③コミット
git commit -m "llama2初回コミット"


------------------------------------------------------------
３）リモートリポジトリへアップロード
------------------------------------------------------------

※後からローカル/リモートを紐づける場合
git remote add origin https://github.com/K1-42/python-ai.git

git push origin main
